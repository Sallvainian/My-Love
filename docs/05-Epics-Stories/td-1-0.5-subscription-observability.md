## Story

As a test engineer,
I want observable subscription health signals exposed from Supabase Realtime connections,
so that I can write deterministic E2E tests for subscription reconnection and message delivery.

2. Love Notes subscription integrated with health hook
3. Connection state transitions observable
4. E2E test fixtures support mocking subscription drops/reconnects
5. No production UI required
