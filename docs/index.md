## Project Overview

- **Type:** Monolith Web SPA (Progressive Web App)
- **Primary Language:** TypeScript 5.9.3
- **Framework:** React 19.2.3 + Vite 7.3.1
- **Architecture:** Component-based with layered architecture (offline-first)
- **Backend:** Supabase (PostgreSQL + Auth + Storage + Realtime + Edge Functions)
- **Hosting:** GitHub Pages — https://sallvainian.github.io/My-Love/

### Quick Reference

- **Tech Stack:** React 19 / TypeScript 5.9 / Vite 7 / Zustand 5 / Supabase / Tailwind 4
- **Entry Point:** `index.html` → `src/main.tsx` → `src/App.tsx`
- **Architecture Pattern:** Component-based layered SPA with offline-first data strategy
- **State:** 10 Zustand slices, LocalStorage + IndexedDB persistence
- **Database:** 11 Supabase tables (RLS), 8 IndexedDB stores (v5)
- **Node:** 24.x (pinned via .nvmrc) | **Package Manager:** npm

## Getting Started

```bash
git clone https://github.com/sallvainian/My-Love.git
cd My-Love
nvm use          # Node 24.x
npm install
npm run dev      # Starts Vite dev server
```
